# AI-Powered Study Buddy

## Problem Statement
Students often struggle to understand complex concepts during exam time. Online resources are lengthy and teachers may not always be available. This project provides a quick, AI-assisted study tool for revision.

## Solution
AI-Powered Study Buddy explains topics in simple terms, summarizes study notes, generates flashcards, and creates quizzes to support exam preparation.

## Features
- Exam-focused note summarization
- Instant question answering
- Flashcards for quick revision
- Quiz generation with difficulty levels

## Dataset
SQuAD (Stanford Question Answering Dataset) from Kaggle.

## Tech Stack
- Python
- Pandas
- NLTK
- Streamlit

## Outcome
The tool helps students revise faster and focus on important concepts during exams.
